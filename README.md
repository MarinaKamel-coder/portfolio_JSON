# 👩‍💻 Portfolio JSON

Bienvenue sur mon portfolio interactif ! Ce projet présente mon parcours, mes compétences et mes projets sous forme de données JSON, offrant ainsi une présentation dynamique et flexible.

---

## 🛠️ Technologies utilisées

- HTML
- CSS
- JavaScript
- JSON
- Bootstrap

---

## 🚀 Fonctionnalités

- Affichage dynamique des informations à partir de fichiers JSON
- Interface responsive et moderne
- Facilité d'ajout et de mise à jour des données

---

## 📁 Structure du projet

- `index.html` : Page principale du portfolio
- `CSS/` : Dossier contenant les styles CSS
- `JS/` : Dossier contenant les scripts JavaScript
- `data/` : Dossier contenant les fichiers JSON des données
